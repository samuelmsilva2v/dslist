# Projeto DSList
## Sobre o projeto

https://dslist-production-57d8.up.railway.app/

Back-end de uma aplicação construída durante o Intensivão Java Spring, evento organizado pela [DevSuperior](https://devsuperior.com.br/).

A aplicação consiste em duas listas de jogos separadas por gênero, sendo possível ver uma versão resumida das páginas dos jogos e uma versão completa com
descrição, gênero e nota. Além disso é possível mover e trocar os jogos de posição na lista.
